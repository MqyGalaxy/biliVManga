## 介绍

这是一个用于查询 bilibili 有声漫画汉化视频的网站。可以通过查询关键词（译名、要素等）快速搜索近几年各个汉化UP主在B站上分析发布的汉化视频。

网站统计数据来源于B站UP主 [敔信](https://space.bilibili.com/1764312492) 所收集的表格，已获得授权。

## 使用说明

访问 [bilivmanga.moeqy.com](https://bilivmanga.moeqy.com/) 即可食用。

如若小站未更新最新表格数据，请下载UP主 [敔信](https://space.bilibili.com/1764312492) 的最新统计表格，并将其转换为 `.csv` 格式。点击首页右上角的「导入表格」按钮导入 `.csv` 格式的表格。「导入表格」功能基于本地浏览器读取和渲染，并不涉及任何网络传输。

您也可以在网站底部找到原表格的链接（表格更新）跳转查询。

### 本地使用

**Windows**
 1. 需要 `Node.js 22` 及以上的版本并确保安装 `npm` 。
 2. 下载源代码并放置在一个空文件夹内（例如 `D:\biliVManga`）。
 3. 进入该文件夹并点击上方路径栏，输入 `cmd` 或者 `powershell` 打开命令行窗口。
 4. 在窗口中输入 `npm run dev` 。
 5. 等待网页启动，直到弹出本地地址（例如 `https://localhost:4321/`），将其在浏览器中打开即可食用。

### 本地数据更新

**如果您不知道您在做什么，请不要使用！**

原表格数据保存在 `src/data/comics.csv` 。
如果需要更新请下载UP主 [敔信](https://space.bilibili.com/1764312492) 的最新统计表格，并将其转换为 `.csv` 格式。按照以下方法更新：

 1. 覆盖本地位于 `src/data/` 的 `comics.csv`。
 2. 回到根目录打开命令行并输入：
```
node fetchCovers.mjs
```
这个脚本用于通过BV号爬取b站视频封面。为了防止制造大量的流量，脚本每秒只请求 2 次并有 0.5 秒的间隔，这会导致封面下载很慢，**请耐心等待下载完成并且请勿修改这个脚本！**

本地已存在的封面会直接跳过。**一些被隐藏或者下架的视频会下载失败**，这是正常现象。

 3. 脚本提示“全部完成！本次新增下载 [数字] 张封面。”即为完成。
 4. 启动网页检查是否更新完成（可以通过右上角的统计数量来校准）。
